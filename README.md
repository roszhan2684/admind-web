## рҹ§  What is AdMind?

AdMind is a **production-grade AI advertising intelligence platform** that replaces spreadsheets and gut-feeling with data-driven creative decisions. It connects to your ad accounts, scores your creatives in real time, monitors competitors around the clock, and generates campaign briefs in seconds вҖ” all in one luxury-grade UI.

> Built for performance marketers, creative directors, and growth teams who refuse to leave money on the table.

## рҹ—„пёҸ Database Schema

```sql
profiles      вҖ” user identity, company, plan, onboarding state
briefs        вҖ” saved AI-generated creative briefs per user
saved_ads     вҖ” bookmarked ads from the swipe library
projects      вҖ” ad account projects with status management
```

All tables use **Row Level Security** вҖ” users can only ever read/write their own data. A Postgres trigger auto-creates a `profiles` row the instant a new user registers.

```
User enters email
      в”Ӯ
      в–ј

      в”Ӯ
```

- Zero passwords вҖ” OTP only
- Codes expire in **10 minutes**
- Sessions last **1 hour** with refresh token rotation
- Middleware protects all `/app` routes server-side

## рҹҡҖ Getting Started

### Prerequisites
- Node.js 18+
- A [Supabase](https://supabase.com) project
- (Optional) Google Gemini API key for Brief Generator

### 1. Clone and install

```bash
git clone https://github.com/roszhan2684/admind-web.git
cd admind-web
npm install
```

### 2. Set up environment variables

```bash
cp .env.example .env.local
```

Edit `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
NEXT_PUBLIC_API_URL=http://localhost:5050
```

### 3. Set up the database

In your Supabase dashboard вҶ’ SQL Editor, run the full contents of:

```
supabase/setup.sql
```

This creates all tables, RLS policies, and triggers in one shot.

### 4. Configure Supabase Auth

In Authentication вҶ’ URL Configuration:
- **Site URL**: `http://localhost:3000`
- **Redirect URLs**: `http://localhost:3000/**`

### 5. Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
